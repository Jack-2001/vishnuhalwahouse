This repository contains a starter scaffold for a MERN website for Vishnu Halwa House.

Quick commands:

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
```

Set environment variables in `backend/.env` (example):

```
MONGODB_URI=mongodb://localhost:27017/vishnu-halwa
PORT=5000
```

See `progress_document.md` for detailed tasks and installation steps.
